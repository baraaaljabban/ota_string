# Flutter Over The Air String
Update String Over the air but free...

## Installing

#### 1. Get the package
Add this to your pubspec.yaml file:
```
dependencies:
  ota_string: ^0.0.4
  
dev_dependencies:
  ota_generator: ^0.0.1 4
```

#### 2. Install
Install with:
```
flutter pub get
```
